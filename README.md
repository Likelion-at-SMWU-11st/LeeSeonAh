# LeeSeonAh
숙명여대 멋사 아기사자 이선아🦁

<br/>

## 🏔️ 프론트엔드 2주차 과제 ##
<ul>
  <li>현재 날짜와 시간을 알려주는 섹션</li>
  <li>투두리스트 추가 섹션과 할 일, 완료한 일을 보여주는 섹션</li>
  <li> + 버튼을 통해 투두리스트 추가 (엔터 입력시 입력 불가) </li>
  <li> delete 버튼을 통해 투두리스트 삭제 </li>
  <li>체크박스 체크시, 체크된 항목이 '📝 할 일' &rarr; '🎖️ 완료' 로 이동</li>
</ul>
